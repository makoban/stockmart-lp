Operating model

投稿本数より先に、責任と工数を決める

90日実証では、素材収集・在庫確認・制作・承認・投稿・週次分析を一つの流れにします。各工程の担当者と週次工数を開始前に確定します。

  1. 店舗素材・在庫時点商品、価格、数量、売場、確認時刻
  2. 本部企画・承認ブランド、法務、在庫表現を確認
  3. 運用制作・投稿同じ計測コードで配信
  4. 週次分析・配分変更結果を次週の企画へ反映

KPI system

経営・利益・週次行動の3層で測る

目標値は、InstagramインサイトとPOSの基準値を取得し、現状との差分が確認できる形で設定します。

四半期 / North Star SNS施策増分貢献利益

増分販売数量 × 商品1点当たり粗利 − 増分制作費 − 広告費

中間成果 店別・商品別の増分粗利

投稿商品と非投稿商品、同曜日・同時間帯を比較

週次の先行指標 商圏内リーチ・保存・共有

地図遷移、LINE追加、48時間販売消化率＋欠品時間

測定原則：UTMやQRは接点の把握に使いますが、それだけで因果を断定しません。投稿SKU／非投稿SKU、同曜日比較、配信地域差を組み合わせて増分を推定します。

06 / 90-day roadmap

五反田で実証し、川口へ移植する。

開業前から基準値を取り、開業施策の反応と販売を検証し、再利用できる「開業SNSプレイブックv1」にまとめます。

  1. Phase 17/23〜8/5

    五反田の開業前計測を整える

    Instagramインサイト、POS・商品粗利、店別QR、地図リンク、投稿コード、撮影・承認フローを準備。吉祥寺・横浜の基準値も取得します。

    完了条件：開業前の基準値と担当者が確定
  2. Phase 28/6〜8/20

    五反田60％、既存店40％で検証する

    五反田の開業・入荷・売場・スタッフを主軸にしつつ、吉祥寺・横浜も継続。新店効果と通常店の差を比較できる状態を保ちます。

    完了条件：新店と既存店の比較データがそろう
  3. Phase 38/21〜9/15

    開業SNSプレイブックv1をつくる

    5つの企画柱を既存店でも比較し、商品・価値・店・期限・在庫時点の投稿型、素材回収、承認、計測を標準化します。

    完了条件：次の新店へ渡せる運用手順が完成
  4. Phase 49月下旬〜90日

    川口への移植準備と投資判断

    先行指標と増分貢献利益を確認し、条件を満たした企画だけ広告を拡大。川口の素材・導線・計測設計へ移植します。

    完了条件：拡大・継続・停止を決定

川口の開業日が90日後になる場合、この期間の完了点は「移植準備完了」とし、開業後の実績検証は次期計画で行います。
